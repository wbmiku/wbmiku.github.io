分享技术干货，打造技术影响力

by admin
on 12月 27, 2021

区块链：向艺术市场深度推进

9月24日，大陆首家 NFT 画廊“艺数力 NFT 数字画廊”宣布正式落地上海M50消息刚刚传出，京、沪等地诸多媒体即高调反应，网易新闻欣喜称“线上艺术走入生活”；雅昌新闻则郑重评论此举“开启NFT艺术‘新时代’”。

9月5日，全球首家NFT实体画廊“创世艺术画廊”（Start Art Gallery）在香港K11 Musea正式开幕，同时还呈现了首次以教育为目的的多元化NFT艺术主题展。

毕竟，距9月5日中国香港K11MUSEA艺术商场5楼508室诞生“全球首个NFT数字艺术画廊”，历时不足一月；距北京保利6月4日举办2021年春拍首场NFT数字艺术专场《Mstaverse：数字孪生》，7月23日上海嘉禾2021年春拍草间弥生首个NFT项目刚刚过去2、3个月，而距2021年3月11日纽约佳士得网络拍卖艺术家Beeple 作品《每一天：前5000天》（Everydays: The First 5000 Days）以至2017年基于以太坊区块链加密朋克 (CryptoPunk)项目的养成渊源，也仅仅只有3、4年的时间。

问题在于，渊源之后区块链在艺术市场中的张扬区间，初始只有艺术市场中坚力量拍卖市场的集聚热烈，至于从拍卖市场向画廊市场的突起转换，20天中的新锐变化，叫人在目不暇接的同时，突然感悟曾经遭逢争议的区块链技术，不仅的确真实进入艺术市场，而且迅速呈现深度指向的势头，这促使我们不能不对相关问题实施应有的观察与评判。

01

溯源NFT

NFT雏形，最早可以追溯到2012年基于比特币改进的一种P2P网络协议——ColoredCoin（彩色币）。

这是第一个类似 NFT 的通证，彩色币由小面额的比特币组成，它可代表多种资产并具有多样用途，成为NFT技术的雏形。2017年，John 和Matt使用ERC20 通证标准开发了世界上第一个NFT项目加密朋克（CryptoPunk），随后Dapper Labs团队使用非同质化通证的 ERC721 通证标准，开发CryptoKitties（加密猫）项目，使NFT的概念正式确立。

NFT是Non-Fungible Token的缩写，中文称为“非同质化代币”，通常是指开发者在以太坊平台上根据ERC721标准/协议所发行的代币，它是具有不可分割、不可替代、独一无二、可验证、可流通、可交易等特性的数字资产所有权。

作为稀缺的基于区块链的数字资产所有权，NFT拥有标准化、通用性、高流动性、恒定性、不可分割性等重要特征。其所有权的转移通过智能合约进行，而每个所有权转移的过程都记录于区块链中，由于区块链的分布式记账所具备的公开性、可追溯、无法篡改等性质，使NFT具备标准化、通用性、高流动性、恒定性、不可分割性等重要特征。

02

谁在促动NFT与艺术结缘

NFT与艺术的结合，既然源于2017年以太坊中加密朋克（CryptoPunk）的像素头像项目。那么，是谁促动了它与艺术的结缘？

Larva Labs (2005)，《CryptoPunks》，2017年作，非同质化代币(jpg)。各24 x 24像素，共9个加密庞克头像。此作于2021年5月11日在佳士得纽约二十一世纪艺术晚间拍卖中以16,962,500美元成交

窥其流踪，我们认为NFT在2017年艺术领域的表现，主要基于以下四方面因素：

其一，随着区块链技术的日臻完善，支持NFT的各项技术条件已趋向成熟。

其二，包括底层标准、区块链平台、交易市场在内的NFT系统的闭环已逐渐形成，可以为NFT的发布、存证、交易等链条提供通道。

其三，伴随区块链和比特币生长的数字货币交易者及从业者，有基础接受NFT概念的参入。2017年9月4日，国家五部委联合监管中国比特币交易，9月30日，比特币中国宣布关闭交易，至此比特币等海外数字货币在中国境内的交易被极大的限制，而NFT作为非同质化代币，本身也具有货币属性，在比特币等数字货币无法在中国交易后，原有的数字货币从业者、交易者将NFT交易作为数字货币交易的一种延续。

其四，NFT的出现为快速发展的数字艺术提供了确权方案，使其便于收藏并有力量促动日益扩大的交易。正如亚洲最大的加密艺术平台CryptoArt.Ai联合创始人秦建新表示：“NFT对于艺术作品而言，最核心意义是赋予数字艺术资产的稀缺性，令数字艺术资产可以被确权。这实际上解决了从20世纪60年代以来，数字艺术作品难以确权的问题。”

2020年，Super Rare、Rarible、Opensea、CryptoArt.Ai等加密艺术平台的出现加快了NFT的大众化进程，随着参与人数的骤增以及交易量的迅速攀升，让NFT艺术在2020年实现全球范围内的爆发式增长，全球NFT艺术品交易价值超过2.5亿美元，较上一年增长了3倍。因此，NFT业界将2020年视为加密艺术的元年。

NFT在加密市场中的火热，迅速蔓延艺术市场中的二级市场，其标志事件，则是2021年3月11日美国艺术家Beeple的NFT艺术作品《每一天：前5000天》于纽约佳士得春拍所获6930万美元成交记录。这一作品的成交，不仅代表全球在世艺术家第三位拍卖高价记录的诞生，也预示NFT艺术已正式成为拍卖市场的一个重要门类。

佳士得《每一天：前5000天》拍卖伊始，NFT艺术迅速引爆国内各大媒体，中国嘉德、北京保利、北京永乐、上海嘉禾等国内各大拍卖公司均相继开始NFT艺术拍卖的尝试。

上述拍卖公司介入NFT艺术的方式，主要分为两种类型，一种是直接拍卖NFT数字艺术作品，藏家得到的是一条NFT代码；另一种是将实体标的进行NFT存证，NFT随实体标的的交易而附赠。

其中上海嘉禾拍卖公司NFT艺术的拍卖最具代表性，涵盖两种NFT拍卖模式，实体作品的NFT存证尤为中国艺术品市场鉴证问题提供了一种有益的解决方案。

属于第一种模式的“起潮时代—现当代艺术及区块链·数字艺术专场”，嘉禾与NFT加密艺术平台CryptoArt.Ai合作，呈现6件数字艺术家的NFT数字艺术作品。

嘉禾NFT拍卖的第二种模式相较前者更为抢眼，开辟“一念永恒—草间弥生《无限的网》原作及区块链数字存证特场”作为平台，融涵草间弥生两件重要作品《无限的网1959，c1979》与《无线的网(MGPP)》原作，并委托CryptoArt.Ai平台为其进行“NFT赋能”，以 ERC-721智能合约将鉴证材料信息的元数据文件铸造成的一枚NFT，通过 IPFS 分布式存储技术永久保存在全球各节点中。确保这两幅作品的流通及背景资料信息永远不会丢失或篡改。此项NFT存证的珍贵之处，还在于它包含鉴证资料出现的艺术家本人及重要评论家的肖像权。于是通过这些鉴证材料的支持、营养，确保了实体标的的价值，同时也为标的今后的市场记录奠定基础。

正是基于上述功能、优势，《无限的网》+鉴证权益NFT才最终以加佣金后约1.05亿元成交，这枚 NFT 随标的赠送买家。

对比标的中相关作品之前的成交记录以及市场行情，结合市场发展规律分析，如果《无限的网1959，c1979》和《无线的网(MGPP)》两件标的没有NFT存证加持的话，其成交额不会超过8000万元。由此可知，一个NFT存证的价值将近2000万元。

NFT在二级市场的成功使其很快蔓延至一级市场。

2021年9月5日，全球首家NFT实体画廊创世艺术画廊在香港K11 Musea正式开幕，标志NFT艺术与传统一级市场的正式结缘。9月24日，大陆首家NFT实体画廊艺数力落地上海M50，又标志NFT已深入国内一级市场。

后者初衷旨在：期望成为中国最大的艺术MCN公司，经营范围涉及加密艺术创作宣发，版权认定，加密艺术IP衍生品，加密作品交易，艺术家孵化等领域。同时为未来NFT衍生品商店、NFT实物链接工具、NFT经纪人公司等方向谋划、布局。

5月20日，加密艺术家宋婷NFT作品《牡丹亭Rêve之标目蝶恋花—信息科技穿透了「我」》在中国嘉德拍卖行以66.7万元成交，该作品系中国嘉德首次上拍的NFT艺术品。

NFT艺术率先在拍卖市场出现而非画廊市场，其真实根底，与NFT公开、透明、可追溯等相关属性密不可分。

众所周知，拍卖市场成交价格倡扬公开，但一级画廊市场的作品价格却讳莫如深，后者尤与NFT公开透明属性发生抵牾，这催生NFT首先介入拍卖市场，至于目前涉及NFT的画廊，又属新面目的NFT展示、交易平台，其属性与传统画廊存在着明显的差异。

综上所述，NFT艺术与市场结合已日益紧密，触角开始深入一级、二级艺术市场，并有可能未来更广泛的与衍生品市场、IP市场融合，有效改变以往艺术市场的固化格局。

03

步入艺术的NFT价值评估与趋向推测

NFT的出现，首先为当下数字艺术版权领域存在版权管理方式有限、确权难、打击盗版侵权成本高等诸多问题提供了一种解决方案。

其次，NFT将区块链技术中可追溯、无法更改的特性应用于艺术市场的鉴证环节，将对艺术作品，尤其是作者在世的现当代艺术作品的追溯、鉴定、备案、存证等一系列鉴证环节提供有效的技术支持，成为解决艺术市场鉴证问题的新路径。

再次，NFT的出现，还对传统艺术市场的监管模式提供了一种新的思路。可以想象，如果每件二级市场上拍标的都能进行NFT存证，那么其后的交易记录将会公开、透明、可追溯且不可更改，并且能有效制止假拍、劣拍等艺术市场乱象。

未来趋向视野中NFT，业界目前众说纷纭，或认为NFT有望被现实世界更多物品利用，成为现实世界和区块链世界的连接，比如房屋、汽车等物理世界的资产，有可能实现大规模上链，以NFT形式存在和流转，果真如此，这将深刻改变我们的日常生活。

与上一种观点相反，艺术市场专家季涛则认为：“当代数字艺术市场的建立需要NFT加密技术的保驾护航，但艺术市场的成长壮大仍需要数字艺术植根入人心。未来NFT市场将会逐渐以传统艺术市场的补充角色而存在。由于上述种种原因，目前还看不到NFT能够颠覆和取代传统艺术市场的任何可能性。”

基于上述，推测 NFT鉴证确权有可能改变目前艺术市场的监管模式，同时，随着NFT数字艺术的风靡及交易量的不断攀升，NFT艺术市场还有望在原有艺术市场格局中发展成为更独立、更公开透明的市场层级，成为与传统一级、二级市场并驾齐驱的市场层级。

04

趋向艺术市场深度的NFT期待考验

从监管及法律层面观察，国家目前还没有任何针对NFT的监管机构，也未出台NFT交易的相关法律法规，这就导致NFT交易存在交易双方的权利和义务无法明确、得不到法律保障的问题。加之国家明确禁止虚拟货币的发行和交易，NFT是否会步其后尘？应该是所有从业者和收藏者不得不面临的困惑与纠结。

投之于NFT向艺术市场深度的推进，也不无关系。由于NFT目前只限定在拍卖市场与画廊市场，保真、公开、可溯源、无法篡改等属性，目前也主要通过当代艺术主体来加以实现，所以，它所存在的不确定性、可塑性将难免存在，有必要接受艺术市场整体发展规律的调整、检验与监督，借助不断推进的深入实践来矫正、补益与提升。

显然，这与我们借助NFT技术完整助益全部艺术主题的殷切期望，尚存在一定的距离，而基于大力弘扬文化，科学推进新时期中国艺术市场深入发展的目标，更需要经历艰难曲折的诸多探索实践。
